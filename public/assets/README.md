# Product Categories Videos

- `model-01.mp4` … `model-06.mp4` — Videos (same length for synced tab switching)
- `posters/model-01.jpg` … `posters/model-06.jpg` — Poster images (shown before video loads)

Category index maps to file number: first category → model-01, second → model-02, etc.
